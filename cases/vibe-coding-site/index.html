02–04

定结构、填内容、定风格

2

让AI告诉你一个网站有哪些部分

"一个个人网站通常有哪些模块？"——AI列出Hero区、关于我、作品集、时间线、联系方式等。你来选要哪些、不要哪些。

先定骨架再填肉，不要一口气全丢给AI。
3

确定每个模块的内容

逐个模块对齐："关于我写什么？""作品集放哪些？"——把你的真实信息填进去。你喂给AI的信息越多，出来的东西越像你。

❗ 这是最花时间也最值得花时间的一步。每个模块对应你的一部分能力和经验，内容才是让人记住你的东西。
4

让AI学习你的视觉风格

丢你已有的网站/品牌色/喜欢的风格给它看。"参考这个网站的配色和排版风格"——给参考比描述风格有效10倍。

Cola会记住你的品牌色和审美偏好，后续所有设计自动对齐，不用每次重复说。
Tip 05 · 核心方法

分模块搓
每次只做一个部分

先做Hero区 → 让AI一口气出5个方案 → 你选一个 → 优化。再做关于我 → 同样流程。逐个搓完所有模块。

Hero区 关于我 作品集 时间线 联系方式

不要一次让AI写完整个网站。分块做，质量高得多，改起来也不崩。
